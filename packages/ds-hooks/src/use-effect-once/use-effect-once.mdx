`useEffectOnce` is a custom hook that runs an effect only once, when the component mounts. It's a common pattern to run an effect only once, and this hook makes it easy to do so.


<Source code={`
import {useEffectOnce} from '@slabs/ds-hooks';

const Demo = () => {
  useEffectOnce(() => {
    console.log('Running effect once on mount')

    return () => {
      console.log('Running clean-up of effect on unmount')
    }
  });

  return null;
};`
}
