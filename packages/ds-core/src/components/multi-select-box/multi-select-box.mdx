# Made with Combobox
MultiSelectBox is an opinionated component built on top of Combobox component. It has a limited set of features to cover only the basic use cases. If you need more advanced features, you can build your own component with Combobox. You can find examples of custom multi select components on the examples page.

# Usage
-MultiSelect provides a way to enter multiple values. MultiSelect is similar to TagsInput, but it does not allow entering custom values.

# Controlled
MultiSelect value must be an array of string or number, other types are not supported. onChange function is called with an array of strings as a single argument.


# Clearable
Set clearable prop to display the clear button in the right section. The button is not displayed when:

* The component does not have a value
* The component is disabled
* The component is read only

# Controlled And Uncontrolled Search
Both controlled and Uncontrolled search support. For controlled, search you have to pass  **searchValue** and **onSearchValue** props
