# isEmptyObject Function

## Description

The `isEmptyObject` function checks whether the given object is empty or not. It returns `true` if the object is empty, and `false` otherwise.

## Usage

<Source code={`

import { isEmptyObject } from '@slabs/ds-utils';

// Example usage
const emptyObject = {};
const nonEmptyObject = { key: 'value' };

const isEmptyObjectResult1 = isEmptyObject(emptyObject);
// Output: true (emptyObject is an empty object)

const isEmptyObjectResult2 = isEmptyObject(nonEmptyObject);
// Output: false (nonEmptyObject is not an empty object)
`} dark={true} language='tsx'/>
