# getObjectFromArrayByValue Function

## Description

The `getObjectFromArrayByValue` function retrieves the first object in an array of objects where the specified key matches the specified value. It internally uses the `indexOfObjectInArray` function to find the index of the object and returns the object if found, or `null` if not found.
