# pickKeys Function

## Description

The `pickKeys` function is used to pick specific keys from an object and return a new object with only the specified keys (also works with nested keys).

## Usage


<Source code={`

import { pickKeys } from '@slabs/ds-utils'; 

// Example Usage

const obj = { a: 1, b: 2, c: 3 };
const pickedKeys = pickKeys(obj, ['a', 'b']);
// Output: { a: 1, b: 2 }

const obj = { a: 1, b: 2, c: { d: 3, e: 4 } };
const pickedKeys = pickKeys(obj, ['a', 'c.d']);
// Output: { a: 1, c: { d: 3 } }

`} dark={true} language='tsx' />
