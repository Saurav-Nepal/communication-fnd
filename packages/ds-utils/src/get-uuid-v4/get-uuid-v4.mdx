# uuidV4 Function

## Description

The `uuidV4` function generates a unique UUID (Universally Unique Identifier) following the version 4 format. It uses a random algorithm to create a string with the pattern 'xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx', where each 'x' is replaced with a random hexadecimal digit, and 'y' is replaced with a random hexadecimal digit from the set [8, 9, A, or B].

## Usage

import { uuidV4 } from '@slabs/ds-utiils';

// Example usage
