# getArrayDifference Function

## Description

The `getArrayDifference` function returns the difference between an array and specified values. It filters out elements from the array that are present in the provided values, resulting in a new array containing only the unique elements.

## Usage


<Source code={`
import { getArrayDifference } from '@slabs/ds-utils';


// Example usage
const originalArray = [1, 2, 3, 4, 5];
const valuesToCompare = [2, 4, 6];

const differenceArray = getArrayDifference(originalArray, valuesToCompare);
// Output: [1, 3, 5]
`} dark={true} language='tsx' />
