# isUndefinedOrNull Function

## Description

The `isUndefinedOrNull` function returns `true` if the given variable is `undefined` or `null`, and `false` otherwise.

## Usage


<Source code={`

import { isUndefinedOrNull } from '@slabs/ds-utils';


// Example usage
const undefinedValue = undefined;
const nullValue = null;
const definedValue = 'Hello, World!';

const isUndefinedOrNullResult1 = isUndefinedOrNull(undefinedValue);
// Output: true (undefinedValue is undefined)

const isUndefinedOrNullResult2 = isUndefinedOrNull(nullValue);
// Output: true (nullValue is null)

const isUndefinedOrNullResult3 = isUndefinedOrNull(definedValue);
// Output: false (definedValue is not undefined or null)
`} dark={true} language='tsx'/>
