# parseToInteger Function
## Description

The `parseToInteger` function parses a string to an integer using the base 10 radix.

## Usage


<Source code={`

import { parseToInteger } from '@slabs/ds-utils';


// Example usage
const numericString = '42';
const parsedInteger = parseToInteger(numericString);
// Output: 42 (parsed integer from the numeric string)
`} dark={true} language='tsx'/>
